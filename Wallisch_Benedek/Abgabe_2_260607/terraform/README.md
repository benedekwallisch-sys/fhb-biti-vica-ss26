# Abgabe 2 – Virtualisierungstechnologien

Diese Abgabe erstellt automatisiert eine virtuelle Maschine auf Exoscale mittels Terraform.

## Funktionen

- Automatische Erstellung einer VM
- Installation eines Webservers mittels CloudInit
- Bereitstellung einer HTML-Webseite
- Bereitstellung eines JSON-API-Endpunkts
- Automatisierte Erstellung und Löschung über GitHub Actions

## Verzeichnisstruktur

- terraform/: Infrastruktur-Code
- cloud-init.yaml.tftpl: VM-Konfiguration
- README.md: Projektbeschreibung
- HERANGEHENSWEISE.md: Dokumentation
